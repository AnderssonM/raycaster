RayCaster API usage

Options Object, display

As seen in previous examples an options object is send to both the CameraCrew class and the RayCaster class on creation. The options object includes lots of settings telling RayCaster and CameraCrew our preferences for how the data should be rendered.

  1. Default Options
       'display': {
           'Dampening': 22.1,
           'Rng_Offset': 0.0,
           'Composition': 0,
           'Camera': 'pCam',
           'Axis Markers': true
       }
