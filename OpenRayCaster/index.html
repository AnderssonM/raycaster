RayCaster Libary

Open Version

RayCaster is a ray tracing volume rendering program implemented for the browser in WebGL/HTML/JavaScript. RayCaster loads binary volumetric data formats, and performs a 3 pass volumetric render.

Features

  • Loads multiple formats (more formats available in closed version)
  • Multiple predefined ColorMaps
  • Color Map scaling
  • Four different composition/integration modes
  • Three different surface modes
  • Filtering function
